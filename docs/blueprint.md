# **App Name**: CBTIS 294 Digital

## Core Features:

- Página Principal Dinámica: Una página de inicio que presente de forma atractiva la información esencial del CBTIS 294, incluyendo enlaces rápidos a las secciones clave y noticias relevantes.
- Explorador de Carreras: Secciones detalladas para cada una de las cuatro carreras (Inteligencia Artificial, Inteligencia de Negocios, Urbanismo, y Cosmetología) con planes de estudio básicos y perfiles de egreso.
- Directorio de Docentes: Listado de la planta docente, organizados por carrera o departamento, incluyendo información relevante de contacto o especialidad.
- Información para Alumnos y Admisiones: Apartado con detalles sobre el proceso de admisión, servicios escolares, y recursos generales para los estudiantes.
- Formulario de Contacto Directo: Un formulario simple para que los interesados puedan enviar preguntas o solicitudes de información directamente a la administración de la escuela.
- Chatbot Educativo Asistido por IA: Un bot conversacional impulsado por IA que actúa como un 'tool' para responder preguntas frecuentes sobre la institución, sus carreras y servicios, mejorando la experiencia del usuario.

## Style Guidelines:

- El color principal es un azul profundo y profesional (#335999), que denota seriedad académica y visión a futuro. Este tono es suficientemente oscuro para contrastar con fondos claros.
- El color de fondo será un azul muy pálido y desaturado (#ECF0F5), casi un blanco grisáceo, para proporcionar un lienzo limpio y fomentar la legibilidad en el entorno educativo.
- Un color de acento azul-cielo brillante y saturado (#4AC4DC) se utilizará para destacar elementos interactivos y captar la atención sin sobrecargar, siendo un análogo vibrante al color principal.
- La tipografía recomendada para títulos y cuerpo de texto es 'Inter', un sans-serif de estilo grotesco que ofrece un aspecto moderno, objetivo y muy legible, ideal para contenido informativo y tecnológico.
- Se utilizarán íconos minimalistas de línea o estilo glifo que refuercen el diseño moderno y la legibilidad de la interfaz, representando claramente las diferentes secciones y servicios.
- El diseño priorizará la claridad y la facilidad de navegación, utilizando un sistema de cuadrícula limpio y jerarquía visual bien definida para guiar al usuario a través de la información escolar.
- Se incorporarán animaciones sutiles y elegantes, como transiciones suaves entre secciones o micro-interacciones al hacer clic, para una experiencia de usuario fluida y sofisticada sin distracciones.
